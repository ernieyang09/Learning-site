# 2778. Sum of Squares of Special Elements

[link](https://leetcode.com/problems/sum-of-squares-of-special-elements/)


```python
class Solution:
    def sumOfSquares(self, nums: List[int]) -> int:
        
        n = len(nums)
        
        nums = [0] + nums
        
        res = 0
        
        for i in range(1, n+1):
            if n % i == 0:
                res += nums[i] ** 2

        return res
```
